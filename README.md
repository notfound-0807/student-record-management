# Student Record Management System (C++)

A console-based mini project built using C++ for managing student records.

## Features
- Add, display, search and delete student records
- File handling using fstream
- Sorting by marks and name
- Top scorer and average marks calculation
- Input validation to avoid infinite loops

## Concepts Used
- C++
- Structures
- File Handling
- STL (vector, sort)
- Functions

## How to Run
1. Compile the code using any C++ compiler
2. Run the executable
3. Follow the menu options
